API Guide and Disclaimers

  • Access currency rates of more than 60 countries. Exchange rates API is a free service for current and historical foreign exchange rates published by the European Central Bank.
  • Associated Disclaimers

APIs

  • Get the latest foreign exchange rates:
    Request URL:
    https://api.exchangeratesapi.io/latest

  • Request specific exchange rates by setting the symbols parameter:
    Request URL:
    https://api.exchangeratesapi.io/latest?symbols=USD,GBP
